## Metodos
A diferencia de lenguajes de programacion como C, no existe un solo tipo y no son siempre funciones. En el caso de Java tenemos dos tipos en los que diferenciar (que a su vez se dividen en dos tipos mÃ¡s).

+ Funcion -> Devuelve un valor, tiene return
```
    public String getModelo() {
        return modelo;
    }
```
+ Procedimiento -> No devuelve nada, no teiene return
```
    public void setModelo(String modelo) {
        this.modelo = modelo;
    }
```
