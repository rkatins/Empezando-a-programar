# Parrafos `<P> ... </P>`
Como podemos intuir un poco por el nombre de la etiqueta, esta hace referencia a 'parrafo', es decir que va a contener texto

Se podria categorizar a esto etiqueta como una de las mas simples, pincipalmente porque solo realiza la accion que podemos considerar mas basica en una p√°gina web, que es mostrar texto.

```
	<p>Soy un parrafo</p>
```
